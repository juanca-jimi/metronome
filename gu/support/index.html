juancajimi@icloud.com

અમે 2 કાર્યકારી દિવસમાં જવાબ આપવાનું લક્ષ્ય રાખીએ છીએ.

વારંવાર પૂછાતા પ્રશ્નો

ક્લિક મોડું કે વહેલું લાગે છે — કેલિબ્રેશન પછી પણ.
Bluetooth હેડફોન કોડેક દ્વારા 150–300 ms ઉમેરે છે, અને એ મૂલ્ય બેટરી અને નજીકતા સાથે બદલાય છે. All Settings → Latency Calibration ચલાવ્યા પછી, તમારા હેડફોન પર સ્પેશિયલ ઓડિયો અને કોઈપણ EQ પ્રીસેટ્સ પણ બંધ કરો — એ ટોચ પર વધારાનું લેટન્સી ઉમેરે છે.
Human Feel સ્વિંગ ટકાવારીથી કઈ રીતે અલગ છે?
સ્વિંગ સબડિવિઝન્સને એક નિશ્ચિત પેટર્ન (લોંગ-શોર્ટ, લોંગ-શોર્ટ)માં ત્રાંસાં કરે છે. Human Feel બધા સબડિવિઝન્સમાં રેન્ડમાઇઝ્ડ માઇક્રો-ટાઇમિંગ ભિન્નતા ઉમેરે છે, એવી રીતે વજન આપીને કે ડાઉનબીટ્સ મજબૂત રહે અને ઓફ-બીટ્સ શ્વાસ લે. સ્ટ્રેન્થ સ્લાઇડર કેટલું એ કંટ્રોલ કરે છે — 100% પર, એ લગભગ એટલું જ છે જેટલું એક સેશન ડ્રમર વિચાર્યા વગર ઉમેરે છે.
શું Ableton Link કાઉન્ટ-ઇન સિંક કરે છે?
Link ફક્ત બીટ ફેઝ અને ટેમ્પો ટ્રાન્સમિટ કરે છે, કાઉન્ટ-ઇન નહીં. જો મેટ્રોનોમ Link સ્રોત હોય, તો તમારું DAW આગલી બીટ સીમા પર જોડાય છે; જો DAW સ્રોત હોય, તો મેટ્રોનોમનું કાઉન્ટ-ઇન બીટ 1 પહેલાં સ્થાનિક રીતે વાગે છે.
શું હું કસ્ટમ ક્લિક સાઉન્ડ ઇમ્પોર્ટ કરી શકું?
હા — Files app દ્વારા ન્યૂનતમ પ્રારંભિક શાંતિ (≤1 ms) સાથે 44.1 kHz પર 16-bit સ્ટીરિયો WAV મેટ્રોનોમના ફોલ્ડરમાં મૂકો, પછી એને Sound Library માં અસાઇન કરો. બીજા ફોર્મેટ પણ વાગશે, પણ વધારાની પ્રારંભિક શાંતિ અનુભવાતા ડાઉનબીટને ખસેડશે.
હું મિશ્ર મીટર, પોલિરિધમ, કે Human Feel કઈ રીતે અનલોક કરું?
એક-વખતની Pro ખરીદી. કોઈ સબસ્ક્રિપ્શન નહીં. અપગ્રેડ શીટ માટે કોઈપણ લોક કરેલા ફીચરને ટેપ કરો, અથવા જો તમે પહેલાં ખરીદ્યું હોય તો Restore Purchases.
હું મારો બધો ડેટા કઈ રીતે ડિલીટ કરું?
All Settings → Delete All Data. પ્રેક્ટિસ લોગ, સ્ટ્રીક્સ, લક્ષ્યો, સેટલિસ્ટ અને ઇમ્પોર્ટ કરેલા સાઉન્ડ ભૂંસી નાખે છે. તાત્કાલિક અને અફર.
શું હું વોલ્યુમ બટન BPM સાથે બાંધી શકું?
હા — All Settings → BPM Controls → Volume Buttons = BPM. ડિફોલ્ટ રૂપે બંધ, જેથી એ સામાન્ય વોલ્યુમ વર્તનને અણધારી રીતે ઓવરરાઇડ ન કરે. ચાલુ હોય ત્યારે, રિંગર વોલ્યુમ પર અસર થતી નથી.
MIDI ક્લોક કે Ableton Link કનેક્ટ થતું નથી.
બંનેને iOS Settings → Metronome માં લોકલ-નેટવર્ક અને Bluetooth પરવાનગીઓ જોઈએ. Link ને એ પણ જરૂરી છે કે બધા ડિવાઇસ એક જ Wi-Fi નેટવર્ક પર હોય — અને એક જ VLAN, જો તમારું રાઉટર ગેસ્ટ-નેટવર્ક આઇસોલેશન વાપરે.

ગોપનીયતા

મેટ્રોનોમ તમારો વ્યક્તિગત ડેટા એકત્ર, વેચતું કે શેર કરતું નથી. એપ તમારી સેટિંગ્સ, પ્રેક્ટિસ લોગ અને સેટલિસ્ટ ફક્ત તમારા ડિવાઇસ પર જ સંગ્રહે છે. વૈકલ્પિક iCloud સિંક તમારા ખાનગી Apple iCloud એકાઉન્ટનો ઉપયોગ કરે છે — ડેવલપર અને કોઈપણ ત્રીજા પક્ષને એની ઍક્સેસ નથી.

માઇક્રોફોન ઍક્સેસનો ઉપયોગ ફક્ત બિલ્ટ-ઇન ટ્યુનર દ્વારા રીઅલ-ટાઇમમાં પિચ શોધવા માટે થાય છે; ઓડિયો ડિવાઇસ પર જ પ્રોસેસ થાય છે અને ક્યારેય રેકોર્ડ કે ટ્રાન્સમિટ થતું નથી.
