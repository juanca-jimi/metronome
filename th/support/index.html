คำถามที่พบบ่อย

เสียงคลิกรู้สึกช้าหรือเร็วเกินไป — แม้จะปรับเทียบแล้วก็ตาม
หูฟัง Bluetooth เพิ่มดีเลย์ 150–300 ms จากตัวถอดรหัส (codec) และค่านี้ยังเปลี่ยนไปตามระดับแบตเตอรีและระยะห่างด้วย หลังจากใช้ All Settings → Latency Calibration แล้ว ควรปิด spatial audio และพรีเซ็ต EQ ใด ๆ บนหูฟังของคุณด้วย — สิ่งเหล่านี้จะเพิ่มความหน่วงซ้อนเข้าไปอีกชั้น
Human Feel ต่างจากค่าเปอร์เซ็นต์ swing อย่างไร?
Swing บิดจังหวะย่อยตามรูปแบบตายตัว (ยาว-สั้น ยาว-สั้น) ส่วน Human Feel เพิ่มความแปรผันจังหวะระดับไมโครแบบสุ่มลงในจังหวะย่อยทั้งหมด โดยถ่วงน้ำหนักให้จังหวะหลัก (downbeat) หนักแน่นมั่นคง ขณะที่จังหวะยก (off-beat) หายใจได้ แถบเลื่อนความเข้มจะควบคุมปริมาณ — ที่ 100% มันก็ราว ๆ สิ่งที่มือกลองรับงานเติมลงไปโดยไม่ต้องคิด
Ableton Link ซิงค์การนับเข้า (count-in) ด้วยไหม?
Link ส่งเฉพาะเฟสของจังหวะและเทมโปเท่านั้น ไม่ได้ส่งการนับเข้า หาก Metronome เป็นแหล่ง Link, DAW ของคุณจะเข้าร่วมในจังหวะถัดไป แต่หาก DAW เป็นแหล่ง การนับเข้าของ Metronome จะเล่นในเครื่องก่อนถึงจังหวะที่ 1
ฉันนำเข้าเสียงคลิกที่กำหนดเองได้ไหม?
ได้ — วางไฟล์ WAV แบบ 16-bit สเตอริโอ 44.1 kHz ที่มีช่วงเงียบนำหน้าน้อยที่สุด (≤1 ms) ลงในโฟลเดอร์ของ Metronome ผ่าน Files app แล้วกำหนดใช้ใน Sound Library ฟอร์แมตอื่นก็เล่นได้ แต่ช่วงเงียบนำหน้าที่มากเกินไปจะทำให้จังหวะหลัก (downbeat) ที่รับรู้เคลื่อนไป
ฉันจะปลดล็อกอัตราจังหวะผสม โพลีริทึม หรือ Human Feel ได้อย่างไร?
ซื้อ Pro ครั้งเดียวจบ ไม่มีการสมัครสมาชิกรายเดือน แตะฟีเจอร์ที่ล็อกไว้เพื่อเปิดแผ่นอัปเกรด หรือกด Restore Purchases หากคุณเคยซื้อไว้แล้ว
ฉันจะลบข้อมูลทั้งหมดของฉันได้อย่างไร?
All Settings → Delete All Data จะล้างบันทึกการซ้อม สถิติต่อเนื่อง เป้าหมาย เซ็ตลิสต์ และเสียงที่นำเข้า ทำทันทีและย้อนกลับไม่ได้
ฉันผูกปุ่มปรับระดับเสียงเข้ากับ BPM ได้ไหม?
ได้ — All Settings → BPM Controls → Volume Buttons = BPM โดยค่าเริ่มต้นจะปิดไว้ เพื่อไม่ให้ไปแทนที่การทำงานปกติของปุ่มปรับเสียงโดยไม่คาดคิด ขณะเปิดใช้งาน ระดับเสียงเรียกเข้าจะไม่ได้รับผลกระทบ
MIDI Clock หรือ Ableton Link เชื่อมต่อไม่ได้
ทั้งสองอย่างต้องได้รับสิทธิ์เครือข่ายภายในและ Bluetooth ใน iOS Settings → Metronome นอกจากนี้ Link ยังกำหนดให้ทุกอุปกรณ์อยู่บนเครือข่าย Wi-Fi เดียวกัน — และ VLAN เดียวกัน หากเราเตอร์ของคุณใช้การแยกเครือข่ายสำหรับผู้เยี่ยมชม (guest network)

ความเป็นส่วนตัว

Metronome ไม่เก็บ ไม่ขาย และไม่แบ่งปันข้อมูลส่วนบุคคลของคุณ แอปจัดเก็บการตั้งค่า บันทึกการซ้อม และเซ็ตลิสต์ของคุณไว้บนอุปกรณ์ของคุณเท่านั้น การซิงค์ iCloud (ตัวเลือกเสริม) ใช้บัญชี Apple iCloud ส่วนตัวของคุณ — นักพัฒนาและบุคคลที่สามใด ๆ ไม่สามารถเข้าถึงได้

การเข้าถึงไมโครโฟนใช้เฉพาะกับเครื่องตั้งสายในตัวเพื่อตรวจจับระดับเสียงแบบเรียลไทม์เท่านั้น เสียงถูกประมวลผลบนอุปกรณ์และไม่มีการบันทึกหรือส่งออกไปที่ใด
