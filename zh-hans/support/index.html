常见问题

即便校准之后，节拍点仍感觉偏晚或偏早。
蓝牙耳机会因编解码器增加 150–300 ms 的延迟，且该数值会随电量与距离而漂移。运行 All Settings → Latency Calibration 之后，请同时关闭耳机的空间音频及任何 EQ 预设 — 它们会叠加额外的延迟。
Human Feel 与摇摆（swing）百分比有何不同？
摇摆按固定模式（长-短、长-短）偏移细分拍。Human Feel 则在所有细分拍上加入随机的微计时变化，并经过加权，使强拍稳如磐石，而弱拍得以呼吸。强度滑块用于控制幅度 — 在 100% 时，大致相当于录音室鼓手不假思索便会加入的那种感觉。
Ableton Link 会同步预备拍（count-in）吗？
Link 只传输节拍相位与速度，不传输预备拍。若节拍器为 Link 源，你的 DAW 会在下一个节拍边界加入；若 DAW 为源，则节拍器的预备拍会在第 1 拍之前于本地播放。
我可以导入自定义节拍音效吗？
可以 — 通过文件 App 将 44.1 kHz、16 位立体声、前导静音极短（≤1 ms）的 WAV 文件放入节拍器的文件夹，然后在 Sound Library 中指定它。其他格式也能播放，但过长的前导静音会使感知到的强拍发生偏移。
我该如何解锁混合拍号、复合节奏或 Human Feel？
一次性 Pro 购买。无订阅。点按任何锁定的功能即可打开升级页面，若你此前已购买，可点按 Restore Purchases。
我该如何删除我的全部数据？
All Settings → Delete All Data。将清除练习记录、连续天数、目标、演出曲目单及已导入的音效。立即生效且不可撤销。
我可以将音量键绑定到 BPM 吗？
可以 — All Settings → BPM Controls → Volume Buttons = BPM。默认关闭，以免意外覆盖正常的音量行为。启用期间，铃声音量不受影响。
MIDI 时钟或 Ableton Link 无法连接。
两者都需要在 iOS Settings → Metronome 中授予本地网络与蓝牙权限。Link 还要求所有设备处于同一 Wi-Fi 网络 — 若你的路由器启用了访客网络隔离，则还需处于同一 VLAN。

隐私

节拍器不会收集、出售或分享你的个人数据。应用仅在你的设备上存储设置、练习记录与演出曲目单。可选的 iCloud 同步使用你私人的 Apple iCloud 账户 — 开发者及任何第三方均无法访问。

麦克风访问权限仅供内置调音器实时检测音高使用；音频在设备端处理，绝不录制或传输。
