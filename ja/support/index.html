よくあるご質問

キャリブレーション後でも、クリックが遅れたり早まったりして聞こえます。
Bluetoothヘッドフォンはコーデックによって150–300 msの遅延が加わり、その値はバッテリー残量や距離によって変動します。All Settings → Latency Calibrationを実行したうえで、ヘッドフォン側の空間オーディオやEQプリセットも無効にしてください — これらはさらに遅延を上乗せします。
Human Feelはスウィングのパーセンテージとどう違うのですか?
スウィングはサブディビジョンを固定パターン(ロング・ショート、ロング・ショート)で偏らせます。Human Feelはすべてのサブディビジョンにランダム化したマイクロタイミングの揺らぎを加え、強拍が揺るがず安定したまま裏拍が息づくように重み付けしています。強度スライダーでその度合いを調整でき、100%ではセッションドラマーが意識せずに加える揺らぎにほぼ相当します。
Ableton Linkはカウントインを同期しますか?
Linkが伝えるのはビートのフェーズとテンポのみで、カウントインは含まれません。メトロノームがLinkのソースの場合、お使いのDAWは次のビートの境界で参加します。DAWがソースの場合は、メトロノームのカウントインがビート1の前にローカルで再生されます。
カスタムのクリック音をインポートできますか?
はい — 16ビットステレオ・44.1 kHz、先頭の無音を最小限(≤1 ms)にしたWAVを、Files app経由でメトロノームのフォルダに入れ、Sound Libraryで割り当ててください。その他の形式も再生できますが、先頭の余分な無音があると強拍の聞こえ方がずれてしまいます。
変拍子、ポリリズム、Human Feelはどうやってアンロックしますか?
買い切りのPro購入で、サブスクリプションはありません。ロックされた機能をタップするとアップグレードのシートが表示されます。以前に購入済みの場合はRestore Purchasesをご利用ください。
すべてのデータを削除するにはどうすればよいですか?
All Settings → Delete All Dataから行えます。練習ログ、連続記録、目標、セットリスト、インポートした音源が消去されます。即時かつ取り消しできません。
音量ボタンをBPMに割り当てられますか?
はい — All Settings → BPM Controls → Volume Buttons = BPMで設定できます。通常の音量操作を予期せず上書きしないよう、デフォルトではオフです。有効な間も、着信音の音量には影響しません。
MIDIクロックまたはAbleton Linkが接続できません。
どちらもiOS Settings → MetronomeでローカルネットワークとBluetoothの許可が必要です。Linkはさらに、すべてのデバイスが同じWi-Fiネットワーク上にあること — ルーターがゲストネットワークの分離を使用している場合は同じVLAN上にあることも必要です。

プライバシー

メトロノームは、お客様の個人データを収集、販売、共有することはありません。アプリは、設定、練習ログ、セットリストをお客様のデバイス上にのみ保存します。任意のiCloud同期は、お客様のプライベートなApple iCloudアカウントを使用し、開発者もいかなる第三者もアクセスできません。

マイクへのアクセスは、内蔵チューナーが音程をリアルタイムで検出するためだけに使用されます。音声はデバイス上で処理され、録音や送信は一切行われません。
