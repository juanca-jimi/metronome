常見問題

節拍聲聽起來偏晚或偏早 — 即使校準後仍是如此。
藍牙耳機會因編解碼器增加 150–300 ms 的延遲，且此數值會隨電量與距離而漂移。執行 All Settings → Latency Calibration 之後，也請關閉耳機上的空間音訊與任何 EQ 預設 — 這些都會額外疊加延遲。
Human Feel 與搖擺（swing）百分比有何不同？
搖擺會以固定模式（長—短、長—短）偏移各個細分音符。Human Feel 則在所有細分音符上加入隨機的微節奏變化，並經過加權，讓強拍穩如磐石，弱拍則自然呼吸。強度滑桿可調整幅度 — 設為 100% 時，效果約等同錄音室鼓手不假思索便會加入的律動。
Ableton Link 會同步預備拍（count-in）嗎？
Link 只會傳輸節拍相位與速度，不會傳輸預備拍。若由節拍器作為 Link 來源，你的 DAW 會在下一個節拍邊界加入；若由 DAW 作為來源，節拍器的預備拍則會在第 1 拍之前於本機播放。
我可以匯入自訂的節拍聲嗎？
可以 — 透過 Files app 將 16 位元立體聲、44.1 kHz、前導靜音極短（≤1 ms）的 WAV 檔放入節拍器的資料夾，再於 Sound Library 中指定使用。其他格式也能播放，但過多的前導靜音會使聽感上的強拍位移。
我要如何解鎖混合拍號、複節奏或 Human Feel？
一次性購買 Pro，無需訂閱。輕觸任何已鎖定的功能即可開啟升級頁面；若你先前已購買，請點選 Restore Purchases。
我要如何刪除所有資料？
All Settings → Delete All Data。會清除練習紀錄、連續紀錄、目標、演出曲目清單與匯入的音效。立即生效且無法復原。
我可以將音量按鈕綁定為 BPM 控制嗎？
可以 — All Settings → BPM Controls → Volume Buttons = BPM。此選項預設為關閉，以免意外覆蓋一般的音量操作。啟用期間，鈴聲音量不受影響。
MIDI 時脈或 Ableton Link 無法連線。
兩者都需要在 iOS Settings → Metronome 中授予區域網路與 Bluetooth 權限。Link 還要求所有裝置連上同一個 Wi-Fi 網路 — 若你的路由器啟用了訪客網路隔離，則須位於同一個 VLAN。

隱私權

節拍器不會收集、販售或分享你的個人資料。App 僅將你的設定、練習紀錄與演出曲目清單儲存在你的裝置上。選用的 iCloud 同步會使用你私人的 Apple iCloud 帳號 — 開發者與任何第三方皆無法存取。

麥克風存取權限僅供內建調音器即時偵測音高之用；音訊會在裝置上處理，絕不會被錄製或傳輸。
